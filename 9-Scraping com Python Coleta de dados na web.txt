Web Scraping é o termo utilizado para definir a prática de coletar automaticamente informações na internet. 
Isto é feito, geralmente, por meio de programas que simulam a navegação humana na web

Beautiful Soup é uma biblioteca Python de extração de dados de arquivos HTML e XML.

O HTML é uma linguagem de marcação formada, composta por tags que determinam o comportamento e disposição de cada elemento do documento,
 e não de programação propriamente dita.
